# Library Management System

## Overview
A backend application that models a library system using object-oriented programming principles.

## Problem
Managing books, users, and borrow/return operations requires a clear and maintainable structure.

## Solution
An object-oriented backend system that separates responsibilities across classes and supports core library operations.

## Key Features
- Book and user management
- Borrow and return operations
- Clear separation of concerns using OOPS principles
- Easily extensible design

## Concepts Used
- Classes and objects
- Inheritance
- Polymorphism
- Abstraction
